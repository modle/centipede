# Running Locally

```
chmod 744 run
./run
```

# Running Jasmine test suite

```
chmod 744 run-jasmine-specs
./run-jasmine-specs
```

# Running Karma coverage tests

```
npm install
chmod 744 run-karma-coverage
./run-karma-coverage
```
